User Guide

TABLE OF CONTENTS

I. INTRODUCTION
The Barangay Residents Information System (BRIS) is designed to help barangay officials manage digitally. It automates resident registration, complaint tracking, and certificate issuance, reducing paperwork and improving service accuracy.

II. SYSTEM OVERVIEW
The system is divided into several modules:
• Home Page - Displays summary of residents, complaints, and certificates.
• Residents - Add, edit, search, or update resident information.
• Complaints - Record and monitor disputes at barangay cases.
• Certificates - Issue Barangay Clearance, Business Clearance, Certificate of Indigency, and Residency.
• Reports - Generate and view barangay data summaries.

III. SYSTEM REQUIREMENTS
• Device: Desktop or Laptop (Windows 10+)

IV. How to Use the System
1. Open the system.
2. Log in using your assigned username and password.
3. Choose a module from the top menu bar.
4. Fill in the needed information and click Save or View.
5. To print certificates, use the View button and confirm details.
6. Use Back or Home to logout when done.

V. Security and Roles
Only authorized users can access the system.
• Admin: Full access and management.
• Secretary: Can view and generate reports.
Always log out after use to protect data.

VI. Troubleshooting
Problem - Solution
• Can’t login → Check username/password
• Data not saving → Ensure all fields are filled and click Save
• Page not loading → Refresh or reopen

VII. Tips
• Review all details before saving or printing.
• Back up data regularly.
• Use the search feature to avoid duplicates.
• Log out after every session.

VIII. Conclusion
The Barangay Residents Information System makes local governance faster and more organized. By automating resident records, complaints, and certificate issuance, barangay staff can provide efficient and transparent service to the community.
